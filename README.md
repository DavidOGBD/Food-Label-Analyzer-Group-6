# Food-Label-Analyzer-Group-6

## Project Description
A Python application that allows users to enter a food product name or barcode, retrieve nutrition information, analyze the food label, identify allergens, suggest healthier alternatives, and save food logs.

## Team Responsibilities

### Member 1
food_product.py
- Create the FoodProduct class
- Store product information

### Member 2
openfoodfacts_client.py
- Connect to Open Food Facts API
- Retrieve product data

### Member 3
nutrition_analyzer.py
- Analyze sugar, fat, salt, and calories

### Member 4
meal_suggestion_generator.py
- Generate healthier alternatives
- Generate recipe suggestions

### Member 5
food_logger.py
- Save food logs to JSON/CSV

### Member 6
validators.py
- Barcode validation using regex

### Member 7
exceptions.py
- Custom exceptions and error handling

### Member 8 (Group Leader)
main.py
- Integrate all modules
- Test the application
- Manage GitHub repository
